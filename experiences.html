Projets académiques

Application de démocratie participative (2024 - en cours)

Ce projet consiste à développer une plateforme open source avec une architecture REST pour faciliter la participation citoyenne. Les utilisateurs pourront proposer des initiatives, participer à des discussions et voter sur des projets, favorisant ainsi une gouvernance plus transparente et inclusive.

Gestion des Jeux Olympiques (2024)

Dans ce projet, une application Java est développée pour gérer les épreuves des Jeux Olympiques. L'application utilise une interface créée avec Penpot, avec une modélisation UML pour structurer les données et une base de données SQL pour gérer les résultats en temps réel et les informations des participants.

Développement Web & Base de Données (2023-2024)

Un site web pour une école fictive a été créé, intégrant la gestion des utilisateurs et des cours via une base de données SQL. Ce projet inclut également un traitement d'images en C++, permettant de gérer et afficher des images liées à l'école et aux étudiants.
